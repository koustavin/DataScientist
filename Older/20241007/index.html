Portfolio

Some of the projects I have recently worked on :

Clustering Indian Districts Based on Census Data

Used 2011 Indian census data to cluster Indian districts based on Decade-on-Decade Population Growth, Sex Ratio, and the Percentage of the Population (male and female) having Steady Employment.

IMDB SQL Data Analysis : Custom Ranking for Movies and Filmmakers

Designing and Optimizing a Full-Sized IMDB Database using only MySQL (no ETL, no code). And then using that database to query custom rankings from IMDB data, like - Top 5 Movies from Each Decade or Determining Ratings and Ranking for Directors from Movie Ratings etc.

Do Stock Markets Go Down on Fridays?

A study of NSE Nifty 50 Index Performance from 2010 to 2023 to verify whether there is any evidence behind the claim that stock Markets Close lower on Fridays.

Fake Football News Classification

Applied extensive Text Preprocessing and several Classification methods (along with detailed Hyperparameter Tuning) on a Football (Soccer) Tweets dataset to achieve very high success rate.
